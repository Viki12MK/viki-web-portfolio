Smart Book Analyzer & Deal Finder

Smart Book Analyzer & Deal Finder is a Python project that combines web scraping with data analysis to uncover the best-value books online. It collects data across multiple pages, converts ratings into numeric scores, and applies a custom “value score” (rating vs price) to rank deals. The project also includes data analysis, export options (CSV/JSON), and optional visualizations and a mini web app for exploring insights—demonstrating end-to-end skills from data collection to actionable insights.

Job Tracker App

Job Tracker App is a full-stack web application designed to help users efficiently manage and track their job applications in one place. Built with Flask, SQLite, and vanilla JavaScript, it allows users to add, update, and organize job entries while tracking their progress through different stages such as Applied, Interviewing, and Offered.
